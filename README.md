## Features

- 箱から出してすぐに反応する
- デスクトップアプリの場合はelectron、モバイルアプリの場合はphonegapとうまく連携します
- 単純な構文
- 拡張可能、あなたはどれだけ想像することはできません！

Website: https://monogatari.io/
